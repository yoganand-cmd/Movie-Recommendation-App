Description:
A responsive web application that helps users discover popular movies and decide whether to watch them based on ratings and reviews. The app fetches real-time movie data from the TMDb API and displays movie posters, ratings, and overviews with a clean UI and interactive hover effects.

Features:

Fetches trending and popular movies using TMDb API

Search functionality to find movies by title

Color-coded rating system (Green / Orange / Red) for quick decision-making

Smooth hover animation to reveal movie overview

Fully responsive design for mobile and desktop devices

Tech Stack:

HTML5

CSS3 (Flexbox, animations, responsive design)

JavaScript (ES6)

REST API (TMDb)

What I Learned:

API integration and asynchronous JavaScript (fetch, async/await)

DOM manipulation and dynamic UI rendering

Handling user input and form events

Responsive UI design and hover interactions
